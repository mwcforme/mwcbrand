Typography rules

Do

  • Use Oswald 600 for all headlines, uppercase
  • Use Montserrat 400 for body, sentence case
  • Track eyebrows 0.32em, headlines 0.04em, captions 0.18em
  • Maintain a clear hierarchy: eyebrow → headline → sub → body
  • Keep line length 50-75 characters for readability

Don't

  • Use Bebas Neue (legacy reference, no longer in spec)
  • Use em-dashes anywhere in any copy
  • Use italics for emphasis (use color or weight instead)
  • Mix more than two font families on a single surface
  • Track headlines below 0.02em or above 0.08em
