Logo System

One mark. One identity.

The MWC logo is a custom-drawn wordmark. Lowercase "men's" in a heavy rounded display face, with "WELLNESS CENTERS" tracked uppercase below. The lowercase rounded "m" stands alone as the bug for favicons and app icons. These artworks ship as PNG, SVG, and PDF files. Do not retype the brand name in any other font and call it the logo.

Usage Rules

Clear space, minimum size, and color

Clear space

Maintain clear space equal to the cap height of the lowercase "m" on all sides. Never let other elements crowd the mark.

Minimum size

Horizontal wordmark: 120 px / 1.25 inches wide minimum. "m" bug: 16 px / 0.2 inches minimum.

Color rules

Navy on Cream is the default. Cream on Navy for dark surfaces. The two-color navy + orange variant is for promotional moments only. Never rotate, distort, recolor outside the approved palette, or apply effects.

Never substitute

Do not retype "Men's Wellness Centers" in Oswald, Bebas Neue, or any other font as a stand-in for the logo. Always composite the supplied PNG, SVG, or PDF file.

Never alter

Do not recolor via CSS filters, gradients, or overlays. Do not skew, italicize, or stretch the mark. Use the appropriate color variant from the pack.

Bug vs. wordmark

Use the full wordmark wherever it fits. Reserve the "m" bug for favicons, app icons, and tight UI containers below 96 px wide.
